
=== Ensembles des codes source des programmes utilisés dans le Projet ===

[algoreco.py] :
    - Permet la reconnaissance des cartes positionnées devant la PiCam, 
        en utilisant une base de données composée de 'samples' de cartes

[blackjack_t.py] :
    - Il s'agit du code principal contrôlant les phases de jeu de la partie 
        de Blackjack
    - Par soucis de clarté, aucun organigramme pour ce programme n'a été fait, car trop 
        complexe à décrire. Il a donc été commenté et est compréhensible lorsque 
        le lecteur parcourt le code source

[lib.py] :
    - Contient les fonctions permettant d'actionner les moteurs permettant
        la distribution des cartes au fur et à mesure de la partie
        Elle contient aussi les fonctions permettant de créer des barres de chargement
        dans la console
